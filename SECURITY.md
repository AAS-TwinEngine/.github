
# Security Policy

**Please do not disclose security vulnerabilities or potential threats publicly.**

To report a security issue or potential threat, contact:  
**psirt@mm-software.com**

When reporting, please include:
- Affected component and version
- Type of vulnerability or threat (CWE-ID, CVE-ID if available)
- Steps to reproduce or describe the issue
- Impact assessment (if possible)
- Your contact information
